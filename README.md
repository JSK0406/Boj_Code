## 

사용자의 환경에 맞춰 식물의 성장 정보를 제공하는 플랫폼

### ✔ 주요 기능

- 키트에 탑재된 센서를 통해 해당 키트에 등록된 식물에게 필요한 정보들을 주변 환경을 고려하여 제공
- 원격에서 센서를 조작하여 빛 조절 또는 물 주기 가능
- 식물을 키우면서 관리할 수 있는 다이어리를 통해 사진과 기록을 남길 수 있음
- AI 이미지 분류를 통한 식물 검색 기능 제공
